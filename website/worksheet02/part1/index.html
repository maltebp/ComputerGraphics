Layers


Point size

Small drawing program, where you can square points on the screen.
An additional feature for this entire worksheet is the use of "layers". A layer contains its own renderes (point, circle and triangle), and the layers are rendered from bottom to top, so the topmost layer will overwrite all the others.
The layer features (hiding, creating and deleting) are possible because all objects are stored and redrawn every frame. This allows us to dynamically edit objects (in this case its only used for layers).
