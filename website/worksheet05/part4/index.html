Surface Normals
Each vertex has a (surface) normal, which is included in the drawing information after loading an .obj file. I upload this normal as a vertex attribute, so that each vertex has a position, normal and color attribute.
The normal is vertex normal is used to create smooth lighting of the model. Each fragment uses a normal, which has been received from interpolating the vertex normals. This normal is used for the Phong illumination, creating a smooth surface.
